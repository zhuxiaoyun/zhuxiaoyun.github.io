面向对象

  1. 面向对象编程语言的主要特征参考

    1. 封装，对象是封装的最基本单位，面向对象的封装就是把描述同一对象的属性和行为（变量和方法）封装在同一个类中，实现软件的“高内聚，低耦合”，防止程序相互依赖性带来的变动影响
    2. 继承，在已存在的类的基础上，加入新定义的自己的内容，并可以修改原来已存在的内容，同时共享父类的数据和方法
    3. 多态，程序内定义的引用对象所指向的类型和调用引用对象的方法在编程时并不确定，而是在程序运行期间才确定
    4. 抽象，找出事物的相似和共性之处，然后将这些事物归为一个类，这个类只考虑这些事物的相似和共性之处，忽略与目标无关的方面，包括行为抽象和状态抽象两方面，比如，定义一个类

      class person(){String name, int age}

  2. 面向对象编程的三大特性

    1. 封装，隐藏类的内部实现机制，保护内部数据和具体实现，只对外暴露访问方法
      私有化属性
      提供public的set和get方法
    2. 继承，参考，重用父类方法，两个类为IS-A的关系
      构造器，只能被调用，不能被继承，构建过程是从父类“向外”扩散的，会默认调用父类的构造器
      protected关键字，子类可以调用，但应尽量避免
      向上转型
    3. 多态，参考，包含具体示例，程序内定义的引用对象所指向的类型和通过该引用对象发出的方法调用在编程时并不确定，而是在程序运行期间才确定，动态绑定
      只有类的方法有多态，类的成员变量没有多态性
